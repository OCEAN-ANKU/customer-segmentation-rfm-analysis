# Customer Segmentation using RFM Analysis

## Project Overview
This project focuses on customer segmentation using RFM (Recency, Frequency, Monetary) Analysis on online retail transaction data.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Excel
- Power BI

## Project Workflow
1. Data Cleaning using Excel
2. RFM Analysis using Python
3. Customer Segmentation
4. Data Visualization using Power BI
5. Business Insights Generation

## Customer Segments
- High Value Customers
- Regular Customers

## Key Insights
- High Value customers generate maximum revenue.
- Regular customers are highest in count.
- Some customers may need retention campaigns.

## Files Included
- rfm_output.csv
- RFM_Project.ipynb
- RFM_Dashboard.pbix
